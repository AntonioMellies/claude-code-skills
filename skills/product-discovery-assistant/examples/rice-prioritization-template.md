# Template: Priorização RICE de Oportunidades

## Fórmula

**Score RICE = (Reach × Impact × Confidence) / Effort**

| Parâmetro | O que medir | Escala |
|-----------|-------------|--------|
| **Reach** | Quantos usuários são afetados por trimestre | Número real estimado |
| **Impact** | Impacto na métrica principal por usuário | 0.25 = mínimo / 1 = médio / 2 = alto / 3 = massivo |
| **Confidence** | Confiança na estimativa | % (ex: 80% = 0.8) |
| **Effort** | Person-weeks de trabalho total | Número real estimado |

## Critérios de Desempate

Quando dois itens têm RICE parecido, priorizar aquele que:

1. **Valida uma suposição crítica** da Matriz CSD
2. **Desbloqueia outros itens** do Opportunity Tree
3. **É reversível** — mais fácil de corrigir se errar
4. **Tem maior aprendizado** — gera dados para decisões futuras
